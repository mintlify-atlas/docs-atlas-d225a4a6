## Get Purchase Order Audit

<RequestExample>
```bash cURL
curl -X GET "https://api.example.com/api/provisions/purchase/order/audit?TypeFilter=3&Page=1&PagSize=50" \
  -H "Authorization: Basic <base64-credentials>"
```
</RequestExample>

Retrieves purchase order audit data from the Auditoría órdenes de compra grid.

### Endpoint

```
GET /api/provisions/purchase/order/audit
```

### Authentication

This endpoint requires authentication via Basic Authentication.

### Query Parameters

<ParamField query="TypeFilter" type="integer" required>
  Type of filter:
  - `1` = Previous day
  - `2` = Previous month
  - `3` = Current month
  - `4` = Previous year
  - `5` = Specific month and year
</ParamField>

<ParamField query="Month" type="integer">
  Specific month to search. Valid range: 1-12. Required when TypeFilter is 5.
</ParamField>

<ParamField query="Year" type="integer">
  Specific year. Valid values from the previous year onwards. Required when TypeFilter is 5.
</ParamField>

<ParamField query="Page" type="integer" default="1">
  Page number, starts with 1.
</ParamField>

<ParamField query="PagSize" type="integer">
  Number of records to retrieve per page.
</ParamField>

### Response

<ResponseField name="currentPage" type="integer">
  The current page number being returned.
</ResponseField>

<ResponseField name="gridName" type="string">
  Name of the grid (e.g., "AuditoriaOC").
</ResponseField>

<ResponseField name="gridId" type="integer">
  Unique identifier for the grid.
</ResponseField>

<ResponseField name="totalRecordsReturned" type="integer">
  Number of records returned in the current page.
</ResponseField>

<ResponseField name="totalPages" type="integer">
  Total number of pages available.
</ResponseField>

<ResponseField name="totalRecords" type="integer">
  Total number of records matching the query.
</ResponseField>

<ResponseField name="dataRecord" type="object">
  Contains the field definitions and row data.
  
  <Expandable title="dataRecord properties">
    <ResponseField name="fields" type="array">
      Array of field definitions.
      
      <Expandable title="field object">
        <ResponseField name="id" type="integer">
          Field identifier.
        </ResponseField>
        
        <ResponseField name="label" type="string">
          Display label for the field.
        </ResponseField>
        
        <ResponseField name="name" type="string">
          Field name/key.
        </ResponseField>
        
        <ResponseField name="order" type="integer">
          Display order of the field.
        </ResponseField>
        
        <ResponseField name="type" type="string">
          Data type of the field.
        </ResponseField>
        
        <ResponseField name="visible" type="boolean">
          Whether the field is visible by default.
        </ResponseField>
        
        <ResponseField name="width" type="integer">
          Display width of the field.
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="rows" type="array">
      Array of data rows, each represented as a dictionary with field names as keys.
    </ResponseField>
  </Expandable>
</ResponseField>

- **200 OK**: Successfully retrieved purchase order audit data
- **401 Unauthorized**: Missing or invalid authentication token

### Caching

This endpoint is cached for 900 seconds (15 minutes).
