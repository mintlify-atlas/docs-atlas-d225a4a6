## Get Transactions

<api-endpoint method="GET" path="/api/accounting/transactions" />

Retrieves a paginated grid of accounting transactions.

### Authentication

<auth-required />

This endpoint requires authentication via Bearer token.

### Query Parameters

<param-field query="TypeFilter" type="integer" required>
  Tipo de filtro:
  - `1` - PreviousDay (día anterior)
  - `2` - PreviousMonth (mes anterior)
  - `3` - CurrentMonth (mes actual)
  - `4` - LastYear (año anterior)
  - `5` - FullMonthByYear (mes y año en concreto)
  - `6` - Custom
  - `7` - AllRecords
</param-field>

<param-field query="Month" type="integer">
  Mes en concreto a buscar. Valid range: 1-12. Required when TypeFilter is 5 (FullMonthByYear).
</param-field>

<param-field query="Year" type="integer">
  Año en concreto. Valid values start from the previous year. Required when TypeFilter is 5 (FullMonthByYear).
</param-field>

<param-field query="Page" type="integer" default="1">
  Número de página. Starts at 1.
</param-field>

<param-field query="PagSize" type="integer">
  Número de registros a obtener (page size).
</param-field>

### Response

<response-field name="currentPage" type="integer">
  The current page number
</response-field>

<response-field name="gridName" type="string">
  Name of the grid
</response-field>

<response-field name="gridId" type="long">
  Unique identifier for the grid
</response-field>

<response-field name="totalRecordsReturned" type="integer">
  Number of records returned in this page
</response-field>

<response-field name="totalPages" type="integer">
  Total number of pages available
</response-field>

<response-field name="totalRecords" type="integer">
  Total number of records across all pages
</response-field>

<response-field name="dataRecord" type="object">
  Contains the grid data structure with fields and rows
  
  <expandable title="DataRecord Properties">
    <response-field name="fields" type="array">
      Array of field definitions for the grid columns
      
      <expandable title="Field Properties">
        <response-field name="id" type="integer">
          Field identifier
        </response-field>
        
        <response-field name="label" type="string">
          Display label for the field
        </response-field>
        
        <response-field name="name" type="string">
          Field name
        </response-field>
        
        <response-field name="order" type="integer">
          Display order of the field
        </response-field>
        
        <response-field name="type" type="string">
          Data type of the field
        </response-field>
        
        <response-field name="visible" type="boolean">
          Whether the field is visible
        </response-field>
        
        <response-field name="width" type="integer">
          Display width of the field
        </response-field>
      </expandable>
    </response-field>
    
    <response-field name="rows" type="array">
      Array of data rows. Each row is a dictionary with field names as keys.
    </response-field>
  </expandable>
</response-field>

### Example Request

```bash cURL
curl -X GET "https://api.example.com/api/accounting/transactions?TypeFilter=3&Page=1&PagSize=50" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Accept: application/json"
```

```bash Get current month transactions
curl -X GET "https://api.example.com/api/accounting/transactions?TypeFilter=3&Page=1&PagSize=100" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```bash Get specific month and year
curl -X GET "https://api.example.com/api/accounting/transactions?TypeFilter=5&Month=6&Year=2025&Page=1&PagSize=50" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
